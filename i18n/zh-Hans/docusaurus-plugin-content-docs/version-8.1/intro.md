# 介绍
Raaghu 前端套件是使用 React 18 和 Raaghu 设计系统构建的。它提供了一个健壮的、模块化的架构，完全兼容 ABP 商业版本，提供了生产就绪的最终用户应用程序页面、可定制的仪表板模板，包含约30个应用程序页面、预定义的主题、模块和端到端文档。

## 入门
本教程指导如何使用启动模板创建和运行 Web 应用程序。

## UI 框架
Raaghu 前端套件与 ABP 商业版本及其各自的发布版本兼容。
