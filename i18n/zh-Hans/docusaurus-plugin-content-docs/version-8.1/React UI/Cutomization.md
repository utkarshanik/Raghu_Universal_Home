# 定制

**自定义**

### 管理个人资料页面选项卡

在单击主机管理员个人资料的下拉菜单后，您将获得4个选项卡，分别是“个人资料图片”、“更改密码”、“个人信息”和“双因素”。

通过 @volo/abp.ng.account/public/config 包公开的 ManageProfileTabsService 可以管理个人资料页面中的选项卡。您可以使用此服务添加、删除或编辑选项卡。
